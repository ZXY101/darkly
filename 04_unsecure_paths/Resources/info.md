# 04_unsecure_paths

Able to navigate through the server files.

## Method

You are able to navigate through the server's files by typing `../` in the url.

You can naviagate to `http://192.168.1.43/index.php?page=../../../../../../../../../../etc/passwd` to get the flag.

## Explanation

Able to access important files on the server, such as the `passwd` file in this case.
