# 01_editable_values

Able to edit a value sent to the backend.

## Method

Go to the Survey Page
Homepage > Survey

Then inspect element and look at one of the selects, you can then edit that options `value` and send any value directly to the backend.

## Explanation

Unintended values could be sent directly to the backend, in this case of a voting system, you could completely break the vote by being to forge a very large amounts of votes.
